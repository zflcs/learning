# A Fast Interrupt Handling Scheme for VLIW Processors

#### 赵方亮笔记

## Introduction

超长指令字 (VLIW) 架构是包含多个功能单元 (FU) 的静态调度架构。

可并行独立执行的操作在编译时确定，而不是在运行时确定。

在发生中断的情况下，处理器状态与乱序执行处理器（包括 VLIW）中的顺序状态不一致，因为指令是乱序完成的。
